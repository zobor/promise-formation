# promise-formation

拓展 Promise.all 的功能，提供 Promise 调用队列

## install

```sh
npm i promise-formation -S
```

**PromiseFormation VS Promise.all**
||Promise.all|PromiseFormation|
|---|---|---|
|并行执行异步函数|✔|✔|
|控制任务顺序|✘|✔|
|限制并发数量|✘|✔|
|动态添加任务|✘|✔|
|动态停止任务|✘|✔|
|停止指定的任务|✘|✔|
